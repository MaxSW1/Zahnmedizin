# Standort-Checkliste: Zahnarztpraxis

## 1. Marktanalyse

### Zahnarztdichte prüfen
- [ ] Zahnarztdichte im Einzugsgebiet recherchieren (Bundesdurchschnitt: ~1.200 Einwohner pro Zahnarzt)
- [ ] Unterversorgung = gut (>1.500 Einwohner pro Zahnarzt)
- [ ] Überversorgung = riskant (<800 Einwohner pro Zahnarzt)
- [ ] KZV (Kassenzahnärztliche Vereinigung) des Bundeslandes nach Bedarfsplanung fragen
- [ ] Altersstruktur der umliegenden Zahnärzte prüfen (viele kurz vor Rente = Übernahmechancen + weniger Wettbewerb bald)

### Bevölkerungsstruktur
- [ ] Einwohnerzahl und Entwicklung (wachsend oder schrumpfend?)
- [ ] Altersstruktur (ältere Bevölkerung = mehr Zahnersatz/Implantate)
- [ ] Kaufkraft / Einkommensniveau (relevant für Privatleistungen und Zuzahlungen)
- [ ] Neubaugebiete geplant? (= zukünftiger Patientenzuwachs)

### Wettbewerb
- [ ] Alle Zahnarztpraxen im 5km-Radius auflisten
- [ ] Deren Leistungsspektrum analysieren (Google, Websites)
- [ ] Google-Bewertungen der Wettbewerber lesen (Schwächen = Chancen)
- [ ] Gibt es bereits Oralchirurgen/MKG-Praxen? (Wenn nein: Alex hat Alleinstellung)
- [ ] Überweiserpotenzial: Wie viele Zahnärzte ohne chirurgisches Angebot gibt es, die Patienten überweisen könnten?

---

## 2. Standort / Immobilie

### Lage
- [ ] **Sichtbarkeit:** Erdgeschoss / Straßenfront bevorzugt (Laufkundschaft, Werbewirkung)
- [ ] **Erreichbarkeit:** ÖPNV-Anbindung, Parkplätze (mindestens 5-10 Patientenparkplätze)
- [ ] **Frequenz:** Lage in Einkaufsstraße, Ärztezentrum oder Wohngebiet?
- [ ] **Barrierefreiheit:** Pflicht bei Neugründung, wichtig für ältere Patienten

### Fläche
- [ ] Mindestens 150-200 m² für 3 Behandlungszimmer
- [ ] Optimal 250-350 m² für 4-5 Behandlungszimmer + Röntgen + OP
- [ ] Raumhöhe mind. 2,50 m (für Deckenleuchten, Klimaanlage)
- [ ] Wasser-/Abwasseranschlüsse vorhanden oder nachrüstbar?
- [ ] Stromversorgung ausreichend? (Röntgen, Kompressor brauchen Starkstrom)
- [ ] Erweiterungsmöglichkeit? (Nachbarfläche anmietbar für Wachstum)

### Mietkonditionen
- [ ] Mietpreis marktüblich? (Vergleich über Immobilienportale)
- [ ] Mietvertragslaufzeit: 10-15 Jahre (Banken wollen lange Laufzeit als Sicherheit)
- [ ] Umbaukosten-Zuschuss vom Vermieter verhandeln
- [ ] Konkurrenzschutzklausel: Keine weitere Zahnarztpraxis im Gebäude/Komplex
- [ ] Nachmieterrecht / Praxisverkaufsklausel (wichtig für Exit)

---

## 3. Praxisübernahme vs. Neugründung

### Checkliste Praxisübernahme

#### Due Diligence
- [ ] BWAs der letzten 3-5 Jahre einsehen
- [ ] Umsatzentwicklung: Steigend, stabil oder fallend?
- [ ] Patientenstamm: Wie viele aktive Patienten (Scheinzahl)?
- [ ] Altersstruktur der Patienten (viele junge = langfristig gut)
- [ ] Privatanteil am Umsatz (höher = besser)
- [ ] Mitarbeiter: Wer bleibt? Qualifikation? Gehälter?
- [ ] Ausstattung: Alter und Zustand der Geräte
- [ ] Mietvertrag: Restlaufzeit, Konditionen, übertragbar?
- [ ] Grund des Verkaufs klären (Alter, Krankheit, Unzufriedenheit?)

#### Bewertung / Kaufpreis
- [ ] Materieller Wert (Geräte, Einrichtung): Gutachter beauftragen
- [ ] Ideeller Wert (Goodwill/Patientenstamm): Üblicherweise 20-50% vom Jahresumsatz
- [ ] **Faustformel Kaufpreis:** 50-100% des Jahresumsatzes
- [ ] Unabhängige Praxisbewertung einholen (z.B. über apoBank, IDZ, oder spezialisierte Berater)

### Checkliste Neugründung
- [ ] KZV-Zulassung beantragen (Vertragszahnarztsitz)
- [ ] Praxis planen mit Dental-Depot (z.B. Henry Schein, Planmeca) – die bieten Komplettplanung an
- [ ] Baugenehmigung / Nutzungsänderung beim Bauamt prüfen
- [ ] Hygiene-Plan erstellen und beim Gesundheitsamt abstimmen
- [ ] Marketing-Budget für Anlaufphase einplanen (höher als bei Übernahme)

## 4. Ausstattung & Technik

### Muss-Ausstattung
- [ ] Behandlungseinheiten (3-5 Stück, ~30-50k pro Einheit)
- [ ] Digitales Röntgen (OPG + Einzelzahnröntgen, ~50-80k)
- [ ] DVT (3D-Röntgen, ~80-150k) – essenziell für Implantologie und Chirurgie
- [ ] Praxissoftware (z.B. Dampsoft, Charly, Z1) inkl. digitale Patientenakte
- [ ] Autoklaven (Sterilisation, ~10-20k)
- [ ] Kompressor + Absauganlage (~15-30k)

### Empfohlene Ausstattung (Differenzierung)
- [ ] Intraoralscanner (z.B. iTero, Primescan, ~30-50k) – weniger Abdrücke, besseres Patientenerlebnis
- [ ] CAD/CAM-Fräseinheit (z.B. CEREC, ~120-160k) – Kronen/Veneers in einer Sitzung
- [ ] OP-Mikroskop für Endodontie (~30-60k)
- [ ] Piezochirurgie-Gerät für Alex (~15-25k)
- [ ] Laser (z.B. Diodenlaser, ~10-20k)
- [ ] Digitale Patientenaufklärung (Monitore in Behandlungszimmern)

### Nicht sofort nötig (Phase 2+)
- [ ] Eigenes Praxislabor
- [ ] Navigierte Implantologie (Guided Surgery)
- [ ] Lachgas-Sedierung / Vollnarkose-Setup

## 5. Personal

### Mindestbesetzung Start (2 Behandler)
- [ ] 2-3 Zahnmedizinische Fachangestellte (ZFA)
- [ ] 1 Empfangskraft / Praxismanagerin
- [ ] 1 Dentalhygienikerin (DH) oder Zahnmedizinische Prophylaxeassistentin (ZMP)
- [ ] Ggf. Auszubildende

### Gehaltsrahmen (Brutto/Monat)
| Position | Gehalt |
|----------|--------|
| ZFA | 2.200-2.800 |
| ZFA mit Weiterbildung | 2.800-3.500 |
| DH / ZMP | 3.000-4.000 |
| Praxismanagerin | 3.000-4.000 |
| Angestellter Zahnarzt | 6.000-10.000 + Umsatzbeteiligung |
| Auszubildende | 900-1.100 |

### Recruiting-Strategie
- [ ] Stellenanzeigen: Dental-Jobportale (z.B. modern-dental-jobs.de), Instagram, lokale Kanäle
- [ ] Employer Branding: Moderne Praxis, gutes Arbeitsklima, Fortbildungsbudget betonen
- [ ] Gehalt leicht über Markt zahlen (5-10% mehr = viel weniger Fluktuation)
- [ ] Praktika/Hospitationen für ZFA-Schüler anbieten (Pipeline aufbauen)

## 6. Marketing & Patientengewinnung

### Vor Eröffnung (3-6 Monate vorher)
- [ ] Praxisname / Branding / Logo entwickeln
- [ ] Website erstellen (mobil-optimiert, Online-Terminbuchung)
- [ ] Google Business Profil anlegen und optimieren
- [ ] Social Media aufsetzen (Instagram, ggf. TikTok)
- [ ] Lokale SEO vorbereiten (Suchbegriffe: "Zahnarzt [Stadt]", "Implantate [Stadt]")
- [ ] Eröffnungskampagne planen

### Laufendes Marketing
- [ ] Google Ads für Neupatienten (~1.000-3.000 EUR/Monat)
- [ ] Bewertungsmanagement: Aktiv Google-Bewertungen sammeln
- [ ] Überweisernetzwerk aufbauen: Andere Zahnärzte, Hausärzte, Kieferorthopäden
- [ ] Recall-System: Automatische Erinnerungen für Kontrolltermine (Patientenbindung)
- [ ] Content-Marketing: Vorher-Nachher-Bilder, Patientengeschichten, Aufklärungsvideos

### Marketing-Budget
| Phase | Monatlich |
|-------|-----------|
| Eröffnung (erste 6 Monate) | 3.000-5.000 |
| Laufend (Standort etabliert) | 1.500-3.000 |
| Pro neuem Standort | 3.000-5.000 |

## 7. Rechtliches & Bürokratie

### Vor Eröffnung
- [ ] Approbation (bei Semmelweis-Abschluss: Anerkennung über Regierungspräsidium)
- [ ] Kassenzulassung bei der KZV beantragen
- [ ] Eintrag ins Zahnarztregister
- [ ] Mitgliedschaft Zahnärztekammer (Pflicht)
- [ ] Berufsgenossenschaft anmelden (BGW)
- [ ] Berufshaftpflichtversicherung abschließen
- [ ] Praxisausfallversicherung
- [ ] Datenschutz (DSGVO): Verarbeitungsverzeichnis, Datenschutzbeauftragter ab 20 MA
- [ ] QM-System einrichten (Pflicht)
- [ ] Röntgenverordnung: Fachkunde-Nachweis, Abnahmeprüfung der Geräte
- [ ] Hygieneplan erstellen und beim Gesundheitsamt vorlegen

### Rechtsform
- [ ] Phase 1: BAG als PartG mbB (Haftungsbeschränkung)
- [ ] Phase 3+: GmbH für MVZ-Struktur
- [ ] Gesellschaftervertrag mit klaren Regelungen: Gewinnverteilung, Entscheidungsrechte, Ausstieg
